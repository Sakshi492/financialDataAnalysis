# sakshiagarwal.dev
  Web/Design Portfolio website

# Install Dependencies
  ```
  npm install
  ```

# Development Server
  ```
  npm run start:flags*
  ```


# Build
  ```
  npm run build:flags*
  ```


> *Set flag as `dev` for Development Environment & `prod` for Production.
